* Single-page application
* Server-side rendering

* NodeJS

* MongoDB

## Installation

```shell
git clone https://github.com/cezerin/cezerin.git cezerin
cd cezerin
npm install
npm run deploy
node index.js
```
Then open [http://localhost:3000](http://localhost:3000) to see your app.

### Requirements

* NodeJS
* NPM
* Mongodb
* Nginx
