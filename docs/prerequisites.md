### Prerequisites

  - **Nginx `>= 1.14.0`**
  ```shell
  apt update && apt install nginx-full
  ```

  - **Mongodb `>= 3.2`**

  - **Node.js `>= v8.11.3`** [Installing Node.js via package manager](https://nodejs.org/en/download/package-manager/#debian-and-ubuntu-based-linux-distributions)

  - **NPM `>= 6.3.0`** 
  ```shell
  npm i -g npm
  ```

  - **PM2 `>= 3.0.3`**
  ```shell
  npm i -g pm2
  ```
